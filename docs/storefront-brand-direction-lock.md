# Storefront Brand Direction Lock

Phase 1 design authority for the Game Time Gift storefront.
This document defines the visual bar for the premium rollout and should be treated as the reference for all future storefront UI decisions.

## Core Direction

The storefront should feel like a luxury retail app, not a sports-themed website template.

The tone is:
- beige
- stone
- black
- soft metallic neutrals
- crisp and modern
- product-forward
- controlled, not decorative

The storefront should not feel:
- navy-dominant
- loud
- candy-colored
- “fan merch”
- overly editorial
- soft in a vague lifestyle way

## What “Nike-Level” Means Here

For this brand, “Nike-level” does not mean copying Nike visuals.
It means matching that level of confidence and finish.

That standard is:
- immediate clarity
- strong hierarchy
- premium restraint
- few colors, used intentionally
- bold, modern typography
- clean product framing
- decisive spacing
- no visual clutter
- no muddy gradients
- no accidental styles

If a section feels generic, theme-like, or overdesigned, it fails the bar.

## Palette Lock

Primary palette:
- canvas: warm ivory
- panels: cream and stone
- text: deep black-brown
- accent: taupe / soft metallic beige
- emphasis: charcoal-black

Approved color behavior:
- large surfaces stay light
- accents are subtle and mostly neutral
- dark colors are used for control, contrast, and confidence
- gradients must stay restrained and tonal

Avoid:
- orange as the dominant emotional color
- pink accents
- saturated gold
- bright blue or royal blue as visual anchors
- rainbow product-page chrome

Allowed:
- navy as a secondary accent for small UI details (icons, dividers, subtle emphasis)

## Typography Lock

Typography should communicate confidence first, elegance second.

Rules:
- primary headlines: bold sans-serif
- supporting statement or premium line: refined serif
- body copy: clean modern sans-serif
- labels, nav, and chips: uppercase or tightly controlled sans-serif

Desired feel:
- bold
- expensive
- readable
- contemporary

Avoid:
- overly romantic serif-heavy layouts
- weak headline contrast
- decorative type for UI controls
- too many font voices on one screen

## Spacing and Rhythm

Spacing should feel deliberate and high-end.

Rules:
- generous outer spacing
- compact internal grouping for controls
- clear separation between product, trust, and story blocks
- consistent radii with no random rounding changes

The experience should feel:
- calm
- structured
- expensive

Not:
- cramped
- floaty
- inconsistent

## Contrast Style

Contrast should come from:
- surface layering
- dark text on light backgrounds
- strong button definition
- restrained shadows

Not from:
- loud color
- hard borders everywhere
- dark slabs across the page

The right balance is soft surfaces with sharp reading contrast.

## Button Treatment

Button language should feel premium and app-like.

Primary buttons:
- dense
- rounded
- slightly weighted
- tactile
- beige / stone with dark text

Secondary buttons:
- light surface
- fine border
- dark text

Avoid:
- bright gold CTA styling
- glossy gradients
- oversized “promo” buttons
- weak button contrast

## Component Tone

Navigation:
- floating and premium
- translucent light shell
- black or near-black utility emphasis

Hero:
- large and bold
- product atmosphere without darkness
- background art should support, never dominate

Cards:
- clean shells
- soft borders
- low-noise surfaces
- clear content hierarchy

Trust strips and chips:
- compact
- premium utility feel
- minimal icon drama

Footer:
- quieter than the hero
- still premium
- no heavy brand-color block

## Non-Negotiables

- Keep the current responsiveness and layout behavior intact
- Navy is allowed as a secondary accent, but must not dominate large surfaces
- Do not drift into orange-led styling
- Do not soften the interface into vague lifestyle minimalism
- Do not use decorative choices that reduce product trust

## Success Test

The homepage passes this phase when:
- it feels premium within three seconds
- the palette reads beige/stone/black, not orange or navy
- the typography feels modern and confident
- the page reads like a polished commerce product
- a picky buyer would trust the brand immediately

## Applies To

This direction should govern:
- homepage
- shop
- product page
- checkout
- consultant landing
- shared navigation
- shared footer
- reusable storefront UI components
